# smart-iot-frontend

智能物联网云平台前端。

`Vue` + `Vuex` + `antd` + `axios`

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
